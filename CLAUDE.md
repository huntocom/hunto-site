# Hunto.com

Read these files before doing anything:
- context.md — what the project is, infrastructure, all key functions
- skills.md — how to edit safely, API patterns, common code snippets
- memory.md — what broke before, decisions made, things to avoid

## Non-negotiable rules
- Always assert find() > 0 before slicing HTML
- Always run div balance check after HTML edits
- Always syntax check JS before deploying
- Never recreate the logo as SVG — embed the actual PNG
- Use URLSearchParams for all catalog API calls, never encodeURIComponent
- Never set src="" with display:block on img tags
